Tentang Text Editor Swift
Text Editor Swift adalah software berbasis console yang berfungsi untuk 
membuka, melihat, dan mengedit file plain text atau teks biasa.


Perintah
Shift + Arrow = memilih data
Ctrl + X = cut/memotong teks yang dipilih
Ctrl + C = copy/menduplikat teks yang dipilih
Ctrl + V = paste/meletakan teks yang dicopy/cut sebelumnya
Ctrl + A = memilih semua teks dalam satu halaman aktif
Ctrl + N = membuat berkas baru/New
Ctrl + O = open/membuka berkas sebelumnya/yang sudah disimpan
Ctrl + S = menyimpan berkas yang sedang aktif
Ctrl + F = menampilkan form pencarian
Ctrl + H = membuka menu help
Ctrl + Q = exit/keluar
